🗜️ Compresión de Matriz

¿Qué hace este programa?

🎯 Objetivo

Leer la matriz fila por fila y contar cuántas veces se repite cada valor consecutivamente. Resultado: lista de tuplas (valor, veces).

📥 Entrada

Una matriz de números (lista de listas).

📤 Salida

Lista de tuplas: [(val, repeticiones), ...]

Ejemplo del enunciado

1
fila 0
1
fila 0
1
fila 0
2
fila 1
2
fila 1
3
fila 1
→
1
× 3 veces
2
× 2 veces
3
× 1 vez

Resultado final: [(1, 3), (2, 2), (3, 1)]

Variables Clave

m

La matriz completa. Se recorre fila por fila.

r

La lista de resultado donde se guardan las tuplas (valor, cuenta).

act

El valor actual que estamos contando. Empieza en None.

cnt

El contador de cuántas veces se repite act de forma consecutiva.

x

El número actual que se está analizando en cada iteración del bucle.

Lógica principal

A
Si act es None (primera vez): guardar x en act y poner cnt = 1.
B
Si x == act: el mismo valor se repite → solo incrementamos cnt += 1.
C
Si x != act: cambió el valor → guardar (act, cnt) en resultado y empezar de nuevo con act = x, cnt = 1.
D
Al finalizar el bucle: siempre guardar el último grupo pendiente.

El Código Completo

def comprimir_matriz(m):
    # r: lista de resultado | act: valor actual | cnt: contador
    r, act, cnt = [], None, 0

    for fila in m:          # recorrer cada fila
        for x in fila:       # recorrer cada número de la fila

            if act is None:  # primer elemento
                act, cnt = x, 1

            elif x == act:   # mismo valor: sumar al contador
                cnt += 1

            else:            # nuevo valor: guardar grupo anterior
                r.append((act, cnt))
                act, cnt = x, 1  # reiniciar con nuevo valor

    # guardar el último grupo (queda pendiente al salir del bucle)
    if act is not None:
        r.append((act, cnt))

    return r

Resumen Visual

¿Por qué guardar al final del bucle?

Cuando termina el bucle, el último grupo nunca se guardó porque no hubo un elemento diferente que lo "disparara". Por eso hay un if act is not None: r.append((act, cnt)) al final.

Ejemplo sin guardar al final ❌

Entrada [1,1,1]: el bucle termina con act=1, cnt=3 pero nunca hay un valor diferente que dispare el append.

Resultado incorrecto: []

Con la línea extra ✅

Al salir del bucle, hacemos r.append((1, 3)) manualmente.

Resultado correcto: [(1, 3)]
