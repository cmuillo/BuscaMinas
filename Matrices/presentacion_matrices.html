¿Qué es una Matriz?

📐 Definición

Una matriz es una tabla de datos organizada en filas (horizontal) y columnas (vertical). En Python se representa como lista de listas.

📍 Acceso a un elemento

Para acceder a la celda de la fila i, columna j:
matriz[i][j]
Las filas y columnas empiezan en índice 0.

🔄 Recorrer la matriz

Se usan dos bucles for anidados: el externo recorre las filas y el interno recorre las columnas.

🌍 Usos prácticos

Imágenes digitales, hojas de cálculo, mapas de videojuegos, tablas de notas, tableros de ajedrez, etc.

# Crear una matriz en Python
matriz = [
    [1,  2,  3,  4],   # fila 0
    [5,  6,  7,  8],   # fila 1
    [9, 10, 11, 12],  # fila 2
]

# Número de filas y columnas
filas    = len(matriz)      # 3
columnas = len(matriz[0])  # 4

# Acceder a la celda [2][3] → 12
print(matriz[2][3])

# Recorrer toda la matriz
for i in range(filas):
    for j in range(columnas):
        print(matriz[i][j])

Ejercicio 1: Rotar imagen 90°

🎯 Problema

Dada una matriz cuadrada N×N (por ejemplo, los pixeles de una foto), devolver una nueva matriz rotada 90° a la derecha (sentido horario).

📷 Caso real

Cuando giras una foto en el teléfono, lo que hace el programa es exactamente esto: rotar la matriz de pixeles.

¿Cómo funciona la rotación?

1
Transponer la matriz: el elemento [i][j] pasa a ser [j][i]. Las filas se convierten en columnas.
2
Invertir cada fila: se le aplica fila[::-1] a cada fila de la transpuesta.
✅
El resultado es la matriz rotada 90° a la derecha. ¡Solo dos pasos sencillos!

💡 Truco para memorizar: "Transponer + Invertir filas = Rotar 90° derecha"

Código — Rotación 90°

# PASO 1 — transponer: transpuesta[j][i] = matriz[i][j]
def transponer(matriz):
    n = len(matriz)
    transpuesta = [[0] * n for _ in range(n)]  # matriz vacía N×N
    for i in range(n):
        for j in range(n):
            transpuesta[j][i] = matriz[i][j]  # intercambia i ↔ j
    return transpuesta

# PASO 2 — invertir cada fila con [::-1] y combinar en una línea
def rotar_90_derecha(matriz):
    return [fila[::-1] for fila in transponer(matriz)]


# ── PRUEBA ─────────────────────────────────────
imagen = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9],
]
rotada = rotar_90_derecha(imagen)
# rotada → [[7,4,1], [8,5,2], [9,6,3]]

🧠 Variables clave

n — tamaño de la matriz (N×N)
transpuesta[j][i] — intercambia índices
[::-1] — invierte una lista en Python

⏱ Complejidad

Recorremos cada celda una vez → O(N²) en tiempo.
Creamos una nueva matriz → O(N²) en espacio.

🔁 Curioso

Si rotas la matriz 4 veces consecutivas, obtienes la matriz original. ¡360° = vuelta completa!

# 1) Promedio por alumno — enumerate recorre FILAS
def promedios_alumnos(matriz, nombres):
    resultado = []
    for i, fila in enumerate(matriz):
        resultado.append((nombres[i], sum(fila) / len(fila)))
    resultado.sort(key=lambda t: t[1], reverse=True)  # mayor a menor
    return resultado

# 2) Promedio por asignatura — zip(*) agrupa COLUMNAS
def promedios_asignaturas(matriz, asignaturas):
    resultado = []
    for j, columna in enumerate(zip(*matriz)):  # zip(*) desempaqueta columnas
        resultado.append((asignaturas[j], sum(columna) / len(columna)))
    resultado.sort(key=lambda t: t[1])  # menor = más difícil
    return resultado

# 3) Nota máxima — doble bucle anidado
def nota_maxima(matriz, nombres, asignaturas):
    max_nota, fila_max, col_max = -1, 0, 0
    for i in range(len(matriz)):
        for j in range(len(matriz[i])):
            if matriz[i][j] > max_nota:
                max_nota, fila_max, col_max = matriz[i][j], i, j
    return max_nota, nombres[fila_max], asignaturas[col_max]

🔑 enumerate()

Itera sobre la lista y da índice + elemento al mismo tiempo. Evita escribir range(len(...)).

🔑 zip(*matriz)

El * "desempaqueta" la lista de filas. zip las agrupa por posición. Resultado: una tupla por cada columna.

🔑 Doble bucle

El bucle for i recorre filas; el bucle for j dentro recorre columnas. Necesario para visitar cada celda.

Índice de celda: fila i, columna j → matriz[i][j]
Transponer: transpuesta[j][i] = matriz[i][j]
Invertir fila: fila[::-1]

Resumen y Trucos de Examen

📌 Crear matriz vacía N×M

NO uses [[0]*M]*N: crea referencias a la misma fila.

    celda = matriz[i][i]

📌 Copiar una matriz (sin referencias)

Checklist para el examen ✅

1
¿La posición [i][j] existe? Verificar 0 ≤ i < filas y 0 ≤ j < columnas.
2
Para recorrer filas: un bucle sobre range(len(matriz)).
3
Para recorrer columnas: añadir segundo bucle sobre range(len(matriz[i])) o usar zip(*matriz).
4
Para rotar: transponer + invertir filas (derecha) o invertir filas + transponer (izquierda).
5
Si tienes que modificar la matriz, trabaja siempre sobre una copia con [fila[:] for fila in matriz].
