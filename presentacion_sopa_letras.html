🔤 Sopa de Letras

¿Qué hace este programa?

🎯 Objetivo

Buscar una palabra dentro de una matriz de letras, moviéndose en 4 direcciones: arriba, abajo, izquierda y derecha.

📥 Entrada

Una matriz de letras y una palabra a buscar.

📤 Salida

True si la palabra existe   False si no existe

Las 4 Direcciones

El algoritmo intenta buscar la palabra empezando en cada celda y avanzando en cada dirección en línea recta sin girar.

⬆️Arriba
(-1, 0)
⬇️Abajo
(+1, 0)
⬅️Izquierda
(0, -1)
➡️Derecha
(0, +1)

¿Cómo funciona el movimiento?

Cada dirección tiene un par (delta_fila, delta_columna). Para avanzar k pasos en esa dirección:

for k, ch in enumerate(palabra):
    ni = fila_inicio + k * delta_fila
    nj = col_inicio  + k * delta_columna

Variables Importantes

m

La matriz completa (lista de listas de letras).

p

La palabra que queremos encontrar.

f, c

Número de filas y columnas de la matriz.

i, j

Posición (fila, columna) desde donde se intenta iniciar la búsqueda.

di, dj

Dirección en la que se avanza (de la lista de 4 posibles).

k, ch

Índice y letra actual de la palabra. Se verifica m[ni][nj] == ch.

ok

Bandera booleana: True si todos los caracteres coincidieron en línea recta.

El Código Completo

def existe_palabra(m, p):
    # Caso especial: palabra vacía
    if not p: return True
    if not m or not m[0]: return False

    f, c = len(m), len(m[0])  # dimensiones
    direcciones = [(-1,0), (1,0), (0,-1), (0,1)]  # 4 dirs

    for i in range(f):
        for j in range(c):              # cada celda
            for di, dj in direcciones:  # cada dirección
                ok = True
                for k, ch in enumerate(p):
                    ni, nj = i + k*di, j + k*dj
                    # fuera de límites O letra no coincide
                    if not (0 <= ni < f and 0 <= nj < c
                            and m[ni][nj] == ch):
                        ok = False; break
                if ok: return True
    return False
