🎨 Flood Fill

¿Qué hace el Flood Fill?

🎯 Objetivo

Igual que el bote de pintura en Paint: rellena con un nuevo valor todo el área conectada que comparte el mismo valor inicial.

📥 Entrada

Una matriz, una posición de inicio (fila, columna) y el nuevo valor a usar.

📤 Salida

Una nueva matriz con el área conectada rellenada con el nuevo valor.

↔️ Conexión

Solo se consideran vecinos en 4 direcciones: arriba, abajo, izquierda, derecha (sin diagonales).

Variables Clave

matriz

La cuadrícula original de valores a rellenar.

fila_inicial, columna_inicial

Posición desde donde empieza el relleno (el "clic" en Paint).

nuevo_valor

El color o número con que se va a rellenar.

valor_inicial

El valor que tenía la celda de inicio. Solo se reemplazan celdas con este valor.

matriz_resultado

Copia de la matriz. Se modifica esta, no la original.

pendientes

Lista tipo pila que guarda las celdas que aún hay que revisar y rellenar.

fila_actual, columna_actual

Celda que se está procesando en cada iteración del bucle.

El Código Completo

def flood_fill_facil(matriz, fila_inicial, columna_inicial, nuevo_valor):

    if not matriz: return []

    filas   = len(matriz)       # cuántas filas tiene la matriz
    columnas = len(matriz[0])  # cuántas columnas tiene

    # Fuera de límites: devolver copia sin cambios
    if fila_inicial < 0 or fila_inicial >= filas or ...:
        return [fila[:] for fila in matriz]

    valor_inicial = matriz[fila_inicial][columna_inicial]

    # Si ya tiene el nuevo valor, nada que hacer
    if valor_inicial == nuevo_valor:
        return [fila[:] for fila in matriz]

    matriz_resultado = [fila[:] for fila in matriz]  # copia
    pendientes = [(fila_inicial, columna_inicial)]      # pila inicial

    while pendientes:
        fila_actual, columna_actual = pendientes.pop()  # saca última

        # Ignorar si está fuera o ya fue cambiada
        if fila_actual < 0 or fila_actual >= filas: continue
        if columna_actual < 0 or columna_actual >= columnas: continue
        if matriz_resultado[fila_actual][columna_actual] != valor_inicial: continue

        matriz_resultado[fila_actual][columna_actual] = nuevo_valor  # rellenar

        pendientes.append((fila_actual - 1, columna_actual))  # arriba
        pendientes.append((fila_actual + 1, columna_actual))  # abajo
        pendientes.append((fila_actual, columna_actual - 1))  # izquierda
        pendientes.append((fila_actual, columna_actual + 1))  # derecha

    return matriz_resultado

Cómo Funciona la Pila

La pila (pendientes) guarda coordenadas. El bucle saca una celda, la rellena y agrega sus vecinos. Se repite hasta que la pila esté vacía.

1
Empieza: pendientes = [(0,0)]. La matriz no ha cambiado aún.
2
Saca (0,0). El valor es 1 = valor_inicial → lo cambia a 9. Agrega (-1,0) (1,0) (0,-1) (0,1).
3
Saca (0,1). Valor 1 → cambia a 9. Agrega vecinos.
4
Si una celda está fuera de límites o tiene valor ≠ 1 (ej: celda con 2) → se ignora con continue.
5
El bucle termina cuando la pila queda vacía. Devuelve matriz_resultado.
