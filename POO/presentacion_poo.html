¿Qué es la Programación Orientada a Objetos?

Es una forma de organizar tu código agrupando datos y las acciones que operan sobre ellos en una misma unidad llamada objeto.

🏗️

Clase

El plano o molde. Define cómo serán los objetos: qué datos tienen y qué pueden hacer.

📦

Objeto

Una instancia de la clase. Es el objeto real creado a partir del molde, con sus propios datos.

🔧

Método

Una función dentro de la clase. Define las acciones que puede realizar el objeto.

El Constructor __init__

Es el primer método que se ejecuta cuando se crea un objeto. Su trabajo es inicializar todos los atributos. self es la referencia al objeto recién creado.

class Membresia:
    def __init__(self, numero, nombre):
        # Datos que recibe el usuario al crear la membresía
        self.numero  = numero        # ej: 101
        self.nombre  = nombre        # ej: "Juan"

        # Contadores que empiezan en 0 automáticamente
        self.sesiones_disponibles      = 0
        self.total_sesiones_compradas  = 0
        self.cantidad_compras          = 0
        self.total_sesiones_utilizadas = 0
        self.cantidad_usos             = 0

        # Estado inicial siempre es "Activa"
        self.estado = "Activa"

Método: comprar_sesiones()

Añade sesiones al saldo. Primero verifica que la membresía esté activa.

def comprar_sesiones(self, cantidad):
    # Paso 1: Verificar estado
    if self.estado == "Suspendida":
        return "Membresía suspendida"

    # Paso 2: Actualizar contadores
    self.sesiones_disponibles     += cantidad
    self.total_sesiones_compradas += cantidad
    self.cantidad_compras         += 1
    return f"Compradas {cantidad} sesiones"

Método: utilizar_sesion()

Descuenta una sesión. Tiene dos validaciones antes de actuar.

def utilizar_sesion(self):
    # Validación 1: membresía activa
    if self.estado == "Suspendida":
        return "Membresía suspendida"

    # Validación 2: hay sesiones disponibles
    if self.sesiones_disponibles == 0:
        return "Sesiones insuficientes"

    # Todo OK: descontar y registrar
    self.sesiones_disponibles      -= 1
    self.total_sesiones_utilizadas += 1
    self.cantidad_usos             += 1
    return "Sesión utilizada ✓"

Métodos sencillos

Estos métodos son cortos pero importantes: consultar estado y cambiar estado.

# Retorna las sesiones actuales
def ver_sesiones_disponibles(self):
    return self.sesiones_disponibles


# Suspende la membresía
def suspender(self):
    self.estado = "Suspendida"
    return "Membresía suspendida."


# Activa la membresía
def activar(self):
    self.estado = "Activa"
    return "Membresía activada."
# Muestra TODA la información
def ver_informacion(self):
    return (
        f"Membresía #{self.numero}\n"
        f"Nombre: {self.nombre}\n"
        f"Estado: {self.estado}\n"
        f"Disp. : {self.sesiones_disponibles}\n"
        f"Usadas: {self.total_sesiones_utilizadas}"
    )

Nivel 3: Menú e Interacción

El programa vive en un bucle while que se repite hasta que el usuario elige "Salir". Las membresías se guardan en una lista de objetos.

# Lista que guarda todos los objetos
membresias = []

while True:          # Repite para siempre
    mostrar_menu()    # Imprime opciones
    opcion = input("Opción: ")

    if opcion == "1":
        crear_membresia(membresias)
    elif opcion == "9":
        break         # Sale del bucle

¿Por qué una lista de objetos?

  • 📋Cada elemento de la lista ES un objeto Membresia
  • 🔍Podemos recorrerla con for m in membresias
  • ➕Añadimos con lista.append(nuevo_obj)
  • 📞Llamamos métodos: m.comprar_sesiones(5)
