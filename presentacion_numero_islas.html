🏝️ Conteo de Islas

¿Cuál es el problema?

🗺️ El mapa

Tenemos una cuadrícula donde cada celda puede ser:
1 = tierra
0 = agua

🏝️ ¿Qué es una isla?

Un grupo de celdas con 1 que están conectadas entre sí (arriba, abajo, izquierda, derecha). Las diagonales no cuentan.

🎯 Objetivo

Contar cuántas islas hay en total en la matriz.

📤 Salida

Un número entero: la cantidad de islas encontradas.

Estrategia: Flood Fill por cada Isla

La idea es recorrer la matriz celda por celda. Cuando encontramos un 1 no visitado, sabemos que es el inicio de una nueva isla. Usamos Flood Fill para marcar toda esa isla como visitada y así no contarla de nuevo.

1
Crear una matriz visitado del mismo tamaño, inicializada en False.
2
Recorrer cada celda (f, c) con dos bucles for.
3
Si matriz[f][c] == 1 y visitado[f][c] == False:
→ Incrementar contador += 1
→ Llamar a inundar() para marcar toda esa isla como visitada.
4
La función inundar() es un Flood Fill: usa una pila y recorre todas las celdas conectadas con valor 1, marcándolas en visitado.
5
Al terminar de recorrer la matriz, devolver contador.

¿Por qué necesitamos visitado?

❌ Sin visitado

Cada celda de una isla se contaría como isla nueva. Una isla de 3 celdas sería contada 3 veces.

✅ Con visitado

Al encontrar el primer 1 de una isla, marcamos todas sus celdas de golpe. Las siguientes celdas de esa isla ya estarán marcadas y se ignorarán.

Variables Clave

matriz

La cuadrícula binaria de entrada. 1=tierra, 0=agua.

visitado

Matriz del mismo tamaño. Guarda True en cada celda que ya fue procesada.

contador

El número de islas encontradas. Aumenta en 1 cada vez que encontramos un 1 no visitado.

f, c

Coordenadas de la celda actual en el bucle principal.

pila

Dentro de inundar(): lista de celdas pendientes por marcar. Se expande a los 4 vecinos.

filas, columnas

Dimensiones de la matriz. Usadas para no salirnos de los límites.

El Código Completo

def contar_islas(matriz):
    if not matriz or not matriz[0]: return 0

    filas    = len(matriz)
    columnas = len(matriz[0])
    visitado = [[False] * columnas for _ in range(filas)]
    contador = 0

    for f in range(filas):
        for c in range(columnas):
            if matriz[f][c] == 1 and not visitado[f][c]:
                contador += 1                   # nueva isla!
                inundar(matriz, visitado, f, c, filas, columnas)

    return contador


def inundar(matriz, visitado, fila, columna, filas, columnas):
    pila = [(fila, columna)]  # empezar desde la celda inicial

    while pila:
        f, c = pila.pop()

        # Ignorar celdas fuera de límites
        if f < 0 or f >= filas or c < 0 or c >= columnas: continue
        # Ignorar agua o ya visitadas
        if visitado[f][c] or matriz[f][c] == 0: continue

        visitado[f][c] = True  # marcar como procesada

        pila.append((f - 1, c))  # arriba
        pila.append((f + 1, c))  # abajo
        pila.append((f, c - 1))  # izquierda
        pila.append((f, c + 1))  # derecha

🎨 Flood Fill

La función inundar() ES Flood Fill. Aquí no cambia colores, sino que marca celdas como visitadas.

🔍 Sopa de Letras

Ambos recorren la matriz buscando un patrón. Aquí el patrón es "grupos de 1s conectados".
